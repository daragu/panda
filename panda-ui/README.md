## Require

- node v16.10.0

  ```bash
  su root
  npm install -g yarn
  npm install -g n
  n v16.10.0
  
  cd /usr/local/n/versions/node
  wget https://repo.huaweicloud.com/nodejs/v16.10.0/node-v16.10.0-linux-x64.tar.gz
  \cp node-v16.10.0-linux-x64/bin/node /usr/local/bin/node /usr/local/bin/node
  ```

## Start ui
```
npm install
npm start
```
